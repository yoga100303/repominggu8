Pengertian CSS

CSS atau Cascading Style Sheets adalah kumpulan perintah yang digunakan untuk menjelaskan tampilan sebuah halaman situs web dalam mark-up language. Salah satu contoh bahasa mereka mark-up paling populer adalah HTML (Hypertext Mark Up Language) yang tergolong sebagai bahasa pemrograman standar dan lazim digunakan dalam pembuatan halaman web. internal css. Seperti namanya, inline CSS adalah kode CSS yang ditulis langsung dalam file HTML.
